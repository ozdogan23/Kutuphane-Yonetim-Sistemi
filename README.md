# 📚 Arem Kütüphane: Kurumsal Web Portalı

Bu proje, bilgisayar mühendisliği eğitimimin ilk yılında, modern web tasarım prensiplerini ve temel backend (PHP) mantığını uygulamak amacıyla geliştirilmiştir.

## 👥 Proje Ortaklığı
Bu çalışma, **Sinem DEMİRTAŞ** ile birlikte bir ekip projesi olarak koordine edilmiştir.

## 📌 Öne Çıkan Özellikler
- **Modüler Arayüz Tasarımı:** Hakkımızda, Popüler Eserler ve Kütüphane Kullanımı gibi alanların blok yapısıyla düzenli sunumu.
- **Yönetici Giriş Sistemi:** Yetkili erişimi için tasarlanmış PHP tabanlı giriş arayüzü.
- **Dinamik Katalog Yapısı:** Popüler eserlerin ve kategorilerin kullanıcıya görsel kartlarla sergilenmesi.
- **Kurumsal Kimlik:** Arem Kütüphanesi için özel renk paleti ve logo çalışmasıyla bütünleşik tasarım.

## 🛠️ Teknik Altyapı
- **Frontend:** HTML5 & CSS3 (Grid/Flexbox mizanpajı)
- **Backend:** PHP (Form yönetimi ve giriş kontrolleri)
